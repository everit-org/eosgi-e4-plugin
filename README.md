### Install
```
$ git clone https://github.com/zsdoma/eosgi-e4-plugin.git
$ cd eosgi-e4-plugin
$ mvn clean install
$ cp plugin/target/org.everit.osgi.dev.e4.plugin-<VERSION>.jar \
<your-eclipse-path>/dropins
```
